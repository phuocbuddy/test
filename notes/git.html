Create local copy of an existing remote repository

git clone git://github.com/phuocbuddy/test.git

Commit changes to local repository

git add <file_name> or <directory_name> git push

The above may display an error

Fix the erro with the second line.

Branch

git checkout -b <branch_name>

Merge

git merge <branch_name>
